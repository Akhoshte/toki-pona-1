# Toki Pona

## nimi lili

### nimi lili insa

* tomo **pi** telo nasa
* mi olin **e** sina.

### nimi lili sinpin

* ona **li** lape.

## nimi suli

* mi

### nimi suli lawa

* **jan** pona

### nimi suli monsi

* jan **pona**

### nimi suli insa

* jan pona **lon** tomo sona

### nimi suli sinpin

* mi **wile** lape.
